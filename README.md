# Command Line Task Manager

## Overview

A simple Command Line Task Manager built in Python using Object-Oriented Programming (OOP). This application allows users to manage their daily tasks directly from the terminal. Tasks are stored in a JSON file, ensuring data persists between program runs.

## Features

* Add new tasks
* View all tasks
* Mark tasks as completed
* Delete tasks
* Save tasks to a JSON file
* Automatically load saved tasks on startup
* Exception handling for invalid inputs

## Technologies Used

* Python
* JSON
* Object-Oriented Programming (Classes and Objects)

## How to Run

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd command-line-task-manager
```

3. Run the program:

```bash
python task_manager.py
```

## Menu Options
